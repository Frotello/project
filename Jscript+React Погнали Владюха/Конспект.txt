Обьявление переменно:
let name = 1;     # Значеник которое подлежит дальнейшей замене
const name = 1;   # Константу так просто не изменить
(старый вариант через var) Ведет к хостингу (всплытие переменной)
"use strict"; Деректива которая пишится вначале файла, для указания того что он выполнен в современном виде

Есть обьекты через: let/const name = {
	name: vlad,
	age: 25, - это свойтвва
	и методы
}

Есть МАСИВЫ (частный случай обьектов) через: let/const name = ['obj1', 'obj2']
для вызова элемента масива name[1(номер от 0 до клнца]
